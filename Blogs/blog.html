Delivered
Al Amin 12:46
Element (like a button) is clicked, an event is directed to the element. If an event handler is set for the element, the event handler is triggered.If a child element of a parent element is clicked, the event will bubble up to the parent element and any event listeners attached to the parent element will be triggered.Then the event "bubbles up" to the elements parent. If an event handler is set for the parent, this event handler is triggered.
